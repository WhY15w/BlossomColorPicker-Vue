# Blossom Color Picker Vue

A Vue.js component for a color picker with a beautiful design and smooth interactions.

Translated from https://github.com/dayflow-js/BlossomColorPicker

> [!TIP]
> The original React version has been upgraded to a framework-free version; please do not use the code in this repository.

demo: https://why15w.github.io/BlossomColorPicker-Vue/

<img src="./img/img.png" alt="Blossom Color Picker Vue" width="400">

## Features

- Beautiful blossom-themed design
- Smooth animations and interactions
- Easy to integrate into Vue 3 projects
- Customizable colors and styles
